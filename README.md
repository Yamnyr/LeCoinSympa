# Le Coin Sympa

## Project Overview
Un projet développé par Quentin Waroquet, composé de deux sous-projets principaux : **LCSFRONT** (React) et **LCSBACK** (Node Express).

## Sous-Projets

### LCSFRONT (Frontend)
- **Technologie** : React
- **Configuration** :
```bash
cd lcsfront
npm install
npm start
```

### LCSBACK (Backend)
- **Technology**: Node.js with Express
- **Setup**:
```bash
cd lcsback
npm install
npm start
```
